Commands

All commands are slash commands - just type / and the command name

♪ Music Commands

/play [song name/url] Play any song or playlist
/pause Pause current track
/resume Resume playback
/stop Stop player and clear queue
/skip Skip current song
/previous Play previous song
/queue View current queue
/nowplaying Show current song
/volume [1-200] Adjust volume
/seek [time] Jump to specific time

☰ Queue Commands

/shuffle Shuffle the queue
/loop Toggle loop (off/track/queue)
/clear Clear the queue
/remove [position] Remove a song from queue
/move [from] [to] Move songs in queue
/jump [position] Jump to a song in queue

♥ Favorites

/savequeue [name] Save current queue
/loadqueue [name] Load a saved queue
/playlists View your saved playlists
/favorite Add current song to favorites
/favorites View your favorite songs
/removefav Remove song from favorites

⚙️ Settings

/settings Configure bot settings
/dj [role] Set DJ role
/prefix [new prefix] Change bot prefix
/help Show help menu
/stats Show bot statistics
/ping Check bot latency
/invite Get bot invite link
